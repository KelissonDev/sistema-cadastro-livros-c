# 📚 Sistema de Cadastro de Livros em C

Este é um sistema simples de cadastro de livros desenvolvido em linguagem **C**, utilizando o **Dev-C++**.
O programa permite cadastrar, pesquisar, editar, listar e ordenar livros por ano de publicação.

## 🚀 Funcionalidades

O sistema possui as seguintes funções:
✔️ Cadastrar livros  
✔️ Listar livros cadastrados  
✔️ Pesquisar livros por código  
✔️ Ordenar livros por ano de publicação  
✔️ Editar dados de um livro  

## 🧠 Estrutura do Sistema

Cada livro possui os seguintes dados:
- Código
- Título
- Autor
- Área
- Ano de publicação
- Editora

O sistema permite cadastrar até **20 livros**.

## 🛠️ Tecnologias Utilizadas

- Linguagem C
- Dev-C++
- Estrutura `struct`
- Vetores
- Laços de repetição
- Ordenação (Bubble Sort)
- Busca sequencial

## ▶️ Como executar o projeto

1. Abra o **Dev-C++**
2. Crie um novo arquivo `.c`
3. Copie o código do projeto
4. Compile e execute o programa

## 🎯 Objetivo

Projeto desenvolvido para praticar lógica de programação, uso de structs e funções em Linguagem C.
